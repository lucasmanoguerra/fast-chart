# Comando: `/agent`

Carga un perfil y dirige la charla a ese rol.

## Uso

```
/agent <nombre>
/agent               # lista
```

## Core

`po` `arqui` `specwriter` `qargento` `devsenior` `revisor` `manteca`

## Quality / design

`architect-guard` `security` `perf` `benchmark` `concurrency` `test-architect` `ux-api` `tech-writer`

## Platform / domain (on-demand)

`data` `cicd` `release` `packaging` `sys-integration` `graphics` `math3d` `numeric` `compliance` `finalgo` `finapi` `devrel`

## Notas

- No existe agente `gui` en la fábrica (UI toolkits son decisión de cada app).  
- Volvés al orquestador con otro comando de flujo (`/status`, `/start-hu`, …).  
- Ver `agents/matrix.md`.
