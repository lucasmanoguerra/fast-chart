# Comando: `/subagent`

Lanza un **worker** de tarea acotada.

## Uso

```
/subagent explore "quién usa Viewport"
/subagent draft-adr "elección de error type"
/subagent run-tests
```

## Defaults

- Read-only salvo que se autorice write path.  
- Prompt autocontenido.  
- Un deliverable.  
- No aprueba gates.

Ver `agents/subagents.md`.
